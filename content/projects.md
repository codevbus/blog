# MetaCareer

[metacareer.pory.app](https://metacareer.pory.app)

MetaCareer is a dead-simple, "Product Hunt"-esque site for sharing content I think is insightful in learning how to "think" about a career in knowledge work. The resources are obviously biased towards careers with exposure to software engineering, but my aim is to find resources that can be applied broadly.

The site is built using [pory.io](https://pory.io), with [airtable](https://airtable.com) as the backend.
